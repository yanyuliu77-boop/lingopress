# LingoPress

每日外刊精读的静态 Web 应用。首篇内容基于用户提供的义乌外刊精读 PDF，包含三种阅读模式、浏览器英文朗读、生词解析与收藏、自动保存笔记、阅读进度、理解测验、连续学习记录和 PWA 离线缓存。

## 访问密钥与店铺自动发货

首批 20 个访问密钥位于未参与部署的 `private/自动发货密钥.csv`。将“网站链接 + 一枚密钥”作为卡密商品导入淘宝或小红书店铺使用的自动发货工具即可。浏览器只接收密钥的 SHA-256 哈希，首次验证后在本机保存授权。

这是一套适合首发的轻量卡密方案。业务规模扩大后，应把校验迁移到服务端数据库，以支持一单一码核销、设备数限制、退款撤销与后台补发。

## 本地运行

```sh
python3 -m http.server 4173
```

打开 `http://localhost:4173`。文章数据目前位于 `app.js` 的 `article` 对象中；新增每日内容时，按同一结构添加文章并由发布流程在每天 07:00 构建即可。
